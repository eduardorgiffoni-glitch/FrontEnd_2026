Brainstorm

O que é Brainstorm?

Brainstorm é uma técnica usada para gerar ideias. Normalmente é feita em grupo, mas também pode ser feita sozinho. O objetivo é pensar em várias ideias sem julgar se elas são boas ou ruins.

Para que serve?

O brainstorm serve para ajudar a resolver problemas, criar projetos e desenvolver a criatividade.

Como fazer um Brainstorm

  1. Escolher um tema ou problema
  2. Anotar todas as ideias que vierem na mente
  3. Não criticar as ideias no começo
  4. Depois organizar e escolher as melhores

Exemplo de Brainstorm

Tema: Criar um aplicativo para estudantes

  • App para organizar tarefas
  • App para fazer resumos
  • App para estudar com flashcards
  • App com videoaulas
